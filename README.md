# GOAT Viz

Interactive visualization project for comparing GOAT (Greatest of All Time) candidates across sports.

## Project Contents

- `src/`: React frontend source code
- `dataset/goat_dataset.json`: core cross-sport dataset
- `GOAT_UI_Plan.md`: full UI and architecture plan

## Prerequisites

- Node.js 18+ (recommended)
- npm

## How to Run

1. Install dependencies:

   ```bash
   npm install
   ```

2. Start the development server:

   ```bash
   npm run dev
   ```

3. Open the local URL shown in the terminal (usually `http://localhost:5173`).

## Build for Production

```bash
npm run build
```

## Preview Production Build

```bash
npm run preview
```
